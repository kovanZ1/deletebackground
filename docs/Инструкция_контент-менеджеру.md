# CaseCutoutTool — инструкция для контент-менеджера

Программа убирает фон с фото чехлов и делает прозрачный PNG. Работает по «маске модели»: один раз настраиваешь маску для модели телефона — дальше все фото этой модели обрабатываются автоматически.

## 1. Установка (один раз)
1. Открой ссылку: https://github.com/kovanZ1/deletebackground/releases/latest
2. Скачай файл **CaseCutoutTool-Setup.exe**.
3. Запусти его, нажми «Далее» → «Установить». На рабочем столе появится ярлык **CaseCutoutTool**.
4. Если Windows покажет «Неизвестный издатель» — нажми «Подробнее» → «Выполнить в любом случае» (для новых программ это нормально).

## 2. Настроить маску модели (один раз на каждую модель чехла)
1. Открой программу, слева выбери **Обучение**.
2. **Открыть фото** → выбери одно чистое фото этой модели.
3. Нажми **Предложить контур** — программа обведёт весь чехол.
4. Нажми **Авто-отверстия** — вырежет все отверстия камеры одним нажатием. Если осталось/лишнее — поменяй ползунок **«Допуск»** и нажми ещё раз, либо поправь Палочкой.
5. Поправь вручную, если нужно:
   - **Палочка** (самое быстрое) — кликни по фону или по отверстию камеры: уберётся вся связная область похожего цвета. Ползунок **Допуск** — насколько широко захватывать (больше = шире).
   - **Кисть** — дорисовать тело чехла (зажми и веди мышью). Кружок-курсор показывает размер.
   - **Ластик** — стереть лишнее (тоже с кружком-курсором).
   - **Вырез** — для круглой дырки: зажми и тяни овал (виден сразу), отпусти — вырезано.
   - **Лассо** — сложная форма: кликами ставь точки по контуру, **двойной клик или правая кнопка** — замкнуть и вырезать.
   - Ползунок **Кисть** — размер кисти/ластика. Кнопки **Undo/Redo** — отменить/вернуть.
5. Внизу впиши **Модель** (например, `iPhone 15 Pro`) и нажми **Сохранить маску**.

## 3. Обработать папку с фото (каждый день)
1. Слева выбери **Авто**.
2. **Папка с фото** — где лежат исходники.
3. **Папка результата** — куда сложить готовые PNG.
4. Выбери **Маску (модель)** или поставь галочку **Автоподбор по имени файла**.
5. Галочки **Создать превью** / **Обрезать по объекту** — по желанию.
6. Нажми **Обработать** и дождись прогресса.

## 4. Проверить результат
- Готовые PNG без фона — в папке результата (по подпапкам моделей).
- Превью на тёмном/светлом/сером фоне — папки `_preview_dark` / `_preview_light` / `_preview_gray`.
- Фото, в которых программа не уверена, — в папке **`_needs_review`** (проверь их руками).
- Таблица по всем файлам — `reports/processing_report.csv`.

## Если что-то не так
- Маска села мимо или захватила лишнее → зайди в **Обучение**, поправь маску этой модели, **Сохрани**, прогони папку заново.
- Прозрачные/стеклянные/глянцевые чехлы пока могут уходить в `_needs_review` — это ожидаемо.
- Принт на фото программа **не трогает** — меняется только фон и отверстия.
